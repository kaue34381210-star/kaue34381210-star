Sou **Analista Fiscal que virou desenvolvedor**. Trabalho na fronteira entre o **fiscal** e a **tecnologia**: transformo rotinas tributárias manuais em ferramentas e automações. Construo **CLIs em Python** para cálculo, validação e conversão de dados fiscais, e também desenvolvo na **web**.

- 🧾 **Fiscal + Tech**: automação de rotinas tributárias e conversão de dados fiscais
- 🐍 **Python**: CLIs para cálculo de impostos, validação de CNPJ/CPF e conversão de NF-e
- 🌐 **Web**: sistemas de gestão com Next.js + Supabase
- 🗄️ **Dados**: SQL, consultas à Receita Federal, tratamento de XML de NF-e
